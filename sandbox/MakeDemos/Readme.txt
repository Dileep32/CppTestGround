This folder contains demo projects for basic usage of make in linux which are described briefly as follows:

simpletest
- very basic Makefile with a single file hello.cpp

maketest
- contains separate .c and .h files

maketest2
- contains separate .c and .h files in different folders

makecpptest
- similar to maketest but with c++

makecpptest2
- similar to maketest2 but contains different folders to keep .cpp and .h files

HelloWorld2
- makefile written in eclipse

HelloWorld3
- makefile written in eclipse that contains a C++ class
